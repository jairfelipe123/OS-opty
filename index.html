Bienvenido a optiOS

optiOS es un sistema operativo simulado con una experiencia visual similar a iOS, diseñado para experimentar una interfaz moderna, fluida y personalizable dentro del navegador.

🔹 Interfaz principal:
Al iniciar, verás una cuadrícula con íconos de aplicaciones como Ajustes, Safari o Calculadora. Toca cualquiera para abrirla. Las animaciones brindan una experiencia de apertura fluida y realista.

🔹 Cierre de aplicaciones:
Para cerrar cualquier app abierta, pulsa la barra inferior (el botón de inicio). Esto te regresará a la pantalla principal.

🔹 Centro de control:
Puedes acceder a funciones rápidas como Wi-Fi, Bluetooth, modo Avión o “No molestar” tocando el ícono de batería en la parte superior derecha. Se abrirá una ventana lateral con controles interactivos.

🔹 Aplicación de Ajustes:
Desde aquí puedes:
- Activar o desactivar el modo oscuro.
- Cambiar el fondo de pantalla por imágenes predeterminadas o cargar una imagen personalizada.
- Entrar a secciones como “General” para ver información del sistema (nombre, versión, etc).
- Acceder a esta guía de ayuda.

🔹 Modo oscuro:
Al activarlo, el sistema cambia automáticamente los colores del fondo y elementos para ofrecerte una experiencia más cómoda en ambientes con poca luz.

🔹 Navegador Safari:
Es un navegador web básico donde puedes ingresar URLs o hacer búsquedas directamente. El sistema detecta si es una dirección o una búsqueda y actúa en consecuencia.

🔹 Calculadora:
Es una calculadora funcional con operaciones básicas (+, –, ×, ÷), porcentaje, cambio de signo, y limpieza de memoria. Funciona como una calculadora real.

🔹 Animaciones:
Cada aplicación y transición está animada para simular la fluidez del sistema operativo real. Puedes usar esto como inspiración para proyectos de front-end o diseño de UX.

Este sistema está en constante evolución. Puedes seguir agregando apps, configuraciones y funciones nuevas para personalizar aún más tu experiencia.

🎉 ¡Explora libremente, experimenta y diviértete con optiOS!
